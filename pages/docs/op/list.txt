====== Data Operation - List ======


===== About =====
''Listing'' is a [[op|data operation]] that shows ''entities'' in a [[docs:resource:tabular|tabular format]]

''Tabulify'' can list:
  * [[docs:resource:resource|data resources]] with the [[docs:tabli:data:list|tabli data list]] command
  * [[docs:connection:connection|connection]] with the [[docs:tabli:connection:list|tabli connection list]] command
  * [[docs:common:conf|configuration]] with the [[docs:tabli:variable:list|tabli conf list]] command
  * [[docs:resource:structure|data structure]] (ie column) with the [[docs:tabli:data:struct|tabli data struct]] command
