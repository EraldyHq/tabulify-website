====== Data Resource - Column ======


===== About =====
A ''column'' is a collection of data with the same [[docs:common:data_type|data type]]. 

<note>
[[tabular|tabular data resource]] are a collection of columns.
</note>

===== Virtual =====
''Tabulify'' supports the addition of column that holds context data via the notion of [[virtual_column|virtual columns]]

===== Data Generation =====
The value of a column may be generated via a [[docs:generator:column|column data generator]]
