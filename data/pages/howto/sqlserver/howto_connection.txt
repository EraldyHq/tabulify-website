====== Sql Server - How-to Connection ======



===== About =====
A [[docs:connection:howtos|default howto sqlserver connection]] is provided after a fresh installation that is used in the [[howto:howto|howto's]]. To use this connection, you need to start the [[#docker|docker image]]



===== Docker =====




==== Download and run the image ====

This connection is based on the [[https://hub.docker.com/r/microsoft/mssql-server|Official Ms SQLServer Docker image]].

  * You can download and start the image with the following command:
<tabs enclosed-tabs>
<panel selected>
   <label>Bash</label>

  * [[docs:common:bash|Bash]]
<code bash>
docker run \
    -e MSSQL_SA_PASSWORD=TheSecret1! \
    -e ACCEPT_EULA=Y \
    -p 1433:1433 \
    -d \
    --name sqlserver \
    mcr.microsoft.com/mssql/server:2022-CU18-ubuntu-22.04
</code>
</panel>
<panel>
   <label>CMD / Powershell</label>
  * [[docs:common:shell|CMD/Powershell]]
<code dos>
docker run ^
    -e MSSQL_SA_PASSWORD=TheSecret1! ^
    -e ACCEPT_EULA=Y ^
    -p 1433:1433 ^
    -d ^
    --name sqlserver ^
    mcr.microsoft.com/mssql/server:2022-CU18-ubuntu-22.04
</code>
</panel>
</tabs>

  * [[docs:tabli:connection:ping|Ping]] your connection to check that the database is up.
<unit>
<code bash>
tabli connection ping sqlserver
</code>
<console>
The connection (sqlserver) has been pinged successfully
</console>
</unit>


==== Start / Stop ====
You can then start and stop the image with the following command:

<code bash>
docker stop sqlserver
docker start sqlserver
</code>
