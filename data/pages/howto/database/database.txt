===== How to follow this howto with another database? =====

This howto are using the [[:docs:system:sqlite:howto|SQlite connection]] as this database does not require to be started.

You can also follow them with any other [[:docs:connection:howtos|howto databases]].

^ System ^ How to start it ^ Connection to use  ^
| Postgres | [[howto:postgres:connection|Start the howto postgres database]] | use ''@postgres'' |
| Mysql | [[:howto:mysql:connection|Start the howto mysql database]] | use ''@mysql'' |
| Oracle | [[:howto:mysql:connection|Start the howto oracle database]] | use ''@oracle'' |

Example:
  * With Sqlite
<code>
tabli data create store_sales@tpcds tpcds_sales@sqlite
</code>
  * This command would become with Postgres
<code bash>
tabli data create store_sales@tpcds tpcds_sales@postgres
</code>
