====== Tabul - Home Directory ======


===== About =====
''Tabulify Home (TABULIFY_HOME)'' is the directory where the [[howto:getting_started:1_install|installation file]] were unzipped.



===== See =====
''tabul'' is able to detect your home. You can see it with the [[docs:tabul:env:list|conf list]] command

<unit>
<code bash>
tabul env list home
</code>
<console>
type       key         value
--------   ---------   ------------------
Internal   home.path   C:\tabul
</console>
</unit>
