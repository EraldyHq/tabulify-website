====== Data Resource - Sql Query (select) ======


===== About =====
A ''Sql Select statement'' or ''Sql query'' is a [[query|query]] performed against a [[docs:system:database|database system]].

A ''Sql Query'' will return [[tabular|tabular data]]
and is therefore considered as a [[content|content data resource]].



===== Attribute =====



^ Name ^ Default ^ Possible Values ^ Description ^
| query-metadata-detection | ''temporary view'' | ''temporary view'', ''runtime'', ''parsing'' | Define the method type to capture the structure if not defined via a data-definition |
