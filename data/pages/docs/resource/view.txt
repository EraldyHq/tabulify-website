====== Data Resource - Sql View ======

===== About =====
''view'' is a [[resource|data resource]] that represents a ''SQL view'' of a [[docs:system:relational|relational database]].


A ''sql view'' can be:
  * [[docs:op:create|created]]
<code bash>
# example creating a view from the select query query_11.sql called query_11 (default)
tabli data create "(sqlite/query_11.sql@tpcds_query)@sqlite"
# example creating a view from the select query query_11.sql called view_query_11
# the type target attribute is mandatory
tabli data create --target-attribute type=view "(sqlite/query_11.sql@tpcds_query)@sqlite" view_query_11@sqlite
# example creating a view from a sql text file
tabli data create --target-attribute type=view sqlite/query_11.sql@tpcds_query view_query_11@sqlite
</code>
  * [[docs:op:drop|dropped]]
  * [[docs:op:transfer|transfered]]


It has the same URI than a [[docs:resource:table|table]]

<code sql>
catalog.schema.view@connection
</code>

===== Media Type =====

The [[docs:resource:type|media type]] for a view is ''sql/view''
