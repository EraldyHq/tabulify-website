====== Data Resource - Selector ======


===== About =====
A ''selector'' is an expression that when matched will select [[resource|resources]] before processing.

There is two kind of selectors:
  * [[data_selector|data selector]] to select [[content|data resources]] or [[container|container resources]]
  * [[script_selector|script selector]] to select [[resource|resources]] that contains [[script|script]]
