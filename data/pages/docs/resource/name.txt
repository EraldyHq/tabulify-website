====== Data Resource - Name ======


===== About =====
''name'' is the full name [[attribute|attribute of a data resource]]. This is also the last part of the [[path|path attribute]].

<note>
[[logicalname|logicalName]] is the name without the file association part.
</note>
===== Operation =====
The [[docs:op:rename|rename operation]] changes the name of a data resource.
