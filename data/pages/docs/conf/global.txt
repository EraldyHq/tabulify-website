====== Global Variable ======





This page is about global [[docs:conf:attribute|variables]]
===== Parameters =====


Global parameters variable are:


|''LOG_LEVEL'' | the [[:docs:common:log|log level]] |
| ''CONF'' | the path to the [[docs:conf:file|configuration file]] |
| ''PROJECT_HOME'' | the [[:docs:connection:project|Project Home path]] |
|''PASSPHRASE'' | the [[:docs:common:passphrase|vault passphrase]] |
| ''ENV'' | the [[:docs:common:environment|execution environment]] |


===== How to =====

==== List global variable ====

With the [[docs:tabli:attribute:list|variable list command]]

==== Set global parameters ====

=== Env ===
With an [[docs:conf:os|operating system variable]].

Example:
<code bash>
TABLI_LOG_LEVEL=warning
</code>

=== Configuration file ===

Most advanced user may edit the [[docs:conf:file|configuration vault]] manually
<code yaml>
globals
  log-level: warning
</code>

but for convenience, we provide the following cli commands:

  * add, remove variables with:
      * [[docs:tabli:attribute:set]]
      * [[docs:tabli:attribute:delete]]
