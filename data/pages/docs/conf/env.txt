====== Env Attributes ======



===== About =====


This page is about the global [[docs:conf:attribute|attributes]] known as ''env''

===== Parameters =====


Global parameters attribute are:


| ''LOG_LEVEL'' | the [[:docs:common:log|log level]] |
| ''CONF'' | the [[docs:conf:vault|configuration vault path]] |
| ''PROJECT_HOME'' | the [[:docs:connection:project|Project Home directory]] |
| ''USER_HOME'' | the [[user_directory|user directory]] |
|''PASSPHRASE'' | the [[:docs:common:passphrase|vault passphrase]] |
| ''ENV'' | the [[:docs:common:environment|execution environment]] |


===== How to =====

==== List global attribute ====

With the [[docs:tabul:env:list|env list command]]

==== Set global parameters ====

=== Env ===
With an [[docs:conf:os|operating system environment variable]].

Example:
<code bash>
export TABUL_LOG_LEVEL=warning
</code>

=== Configuration file ===

Most advanced user may edit the [[docs:conf:vault|configuration vault]] manually
<code yaml>
envs:
  log-level: warning
</code>

but for convenience, we provide also the following cli commands:

  * add, remove attributes with:
      * [[docs:tabul:env:set]]
      * [[docs:tabul:env:delete]]
