====== Os Environment Variables ======


===== About =====
Os Environment Variables can be used:
  * to set the value of [[docs:conf:attribute|attribute]].
  * as [[:docs:conf:template|template value]]




===== Example =====

If you want to set the [[docs:common:environment|execution environment]], you can
set it on Linux as:

<code bash>
env=prod
# or
tabul_env=prod
# or
TABUL_ENV=prod
</code>
