====== Os Environment Variables ======


===== About =====
Os Environment Variables are used to set the value of [[docs:conf:variable|variable]].



It search a variable that starts:

  * with ''tabli''
  * and without

===== Example =====

If you want to set the [[docs:common:environment|execution environment]], you can
set it on Linux as:

<code bash>
env=prod
# or
tabli_env=prod
# or
TABLI_ENV=prod
</code>
