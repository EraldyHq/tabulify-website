====== PostgreSQL ======


===== About =====
''Tabulify'' supports the [[..:relational|relational database]] <icon name="logos:postgresql"/> [[https://www.postgresql.org/|Postgresql]]


<note>
If you don't have any knowledge of ''Tabulify'', you can start with the [[howto:getting_started:getting_started|Learning Guide]].
</note>


===== Connection =====

To create a ''Postgres'' connection, you have the following possibilities:
<tabs skin="tabs">
<tab panel="confVault" selected="true">Configuration Vault</tab>
<tab panel="tabul">Tabul</tab>
</tabs>
<tabpanels>
<tabpanel id="confVault" spacing="m-3">
In a [[docs:conf:vault|Configuration Vault]], to define a Postgres [[docs:connection:connection|connection]] with the name ''myPostgres'', you would write:
\\

<code yaml>
connections:
   myPostgres:
      uri: jdbc:postgresql://host:[port]/[databse]
      user: postgres
      password: ${MY_PASSWORD_ENV}
      driver: org.postgresql.Driver
      natives:
        # any postgres jdbc connection parameters
        ssl: true
</code>
<note>
The [[docs:connection:password|password value]] will be retrieved from the ''MY_PASSWORD_ENV''  [[:docs:conf:os|environment variable]]
</note>
</tabpanel>
<tabpanel id="tabul" spacing="m-3">

With the [[docs:tabul:connection:add|tabul add command of the connection module]], adding a Postgres connection with the name ''postgresConnectionName'':


  * With the Windows CMD console

<code bash>
tabul connection add "postgresConnectionName" "jdbc:postgresql://[host]:[port]/[database]" ^
    --user login ^
    --password password
</code>
  * With Linux Bash
<code bash>
tabul connection add "postgresConnectionName" "jdbc:postgresql://[host]:[port]/[database]" \
    --user login \
    --password password
</code>

</tabpanel>
</tabpanels>

===== Connection Attribute =====
When creating a new [[docs:connection:connection|connection]], you may give the following properties.
==== URL ====

The [[docs:connection:uri|URL]] can take one of this form
<code>
jdbc:postgresql:database
jdbc:postgresql:/
jdbc:postgresql://host/database
jdbc:postgresql://host/
jdbc:postgresql://host:port/database
jdbc:postgresql://host:port/
</code>
where:
  * ''host'' is the host name of the server. Defaults to localhost
  * ''port'' is the port number the server is listening on. Defaults to ''5432''
  * ''database'' is the database name. Defaults to the same name as the user name.


==== Native Attributes ====
All possible [[docs:connection:natives|natives attributes]] (known as connection parameters in Postgres documentation) can be found in the [[https://jdbc.postgresql.org/documentation/use/#connection-parameters|Postgres connection documentation]].

==== Driver ====

We use the driver 42.2.16 that supports PostgreSQL 8.2 or newer

==== Version ====

We support the version supported of the [[https://www.postgresql.org/support/versioning/|postgres versioning scheme]].

For version no more supported, [[:contact|contact us]]

<code sql>
SELECT version();
</code>
