====== Data Operation - List ======


===== About =====
''Listing'' is a [[op|data operation]] that shows ''entities'' in a [[docs:resource:tabular|tabular format]]

''Tabulify'' can list:
  * [[docs:resource:resource|data resources]] with the [[docs:tabul:data:list|tabul data list]] command
  * [[docs:connection:connection|connection]] with the [[docs:tabul:connection:list|tabul connection list]] command
  * [[docs:conf:conf|configuration]] with the [[docs:tabul:env:list|tabul conf list]] command
  * [[docs:resource:structure|data structure]] (ie column) with the [[docs:tabul:data:struct|tabul data struct]] command
