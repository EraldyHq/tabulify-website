====== Data Operation - Create ======

===== About =====
''create'' is an [[op|data operation]] that affects only [[docs:resource:metadata|metadata]].

This operation will create an ''empty'' [[docs:resource:resource|data resource]] and defines optionally its [[docs:resource:metadata|metadata]] (ie [[docs:resource:structure|structure]])


The [[docs:tabul:data:create|data create operation]] supports the ''create'' operation.
