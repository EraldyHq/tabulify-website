====== Column Data Generators ======


===== About =====
A ''column generator'' defines how data is [[generator|generated]] for a column.

They are used in a [[docs:resource:generator|resource generator (aka generator)]] to define the data generated for each columns.



^ Name ^ Description ^
| [[sequence|Sequence]] | Generate data in sequence |
| [[random|Random]] | Generate data randomly |
| [[histogram|Histogram]] | Generate a distribution of data |
| [[docs:generator:expression|Expression]] | Generate derived data (expression from other data) |
| [[docs:generator:entity|Entity]] | Generate data from entity data set |
| [[docs:generator:data-set|Data Set]] | Generate data from a data set |
| [[docs:generator:regexp|Regular Expression]] | Generate data from a regular expression |
